# Trabalho_Final_CG_Carolina_Falcao
# 🏡 Cozy Village 3D - Computação Gráfica

Um diorama interativo 3D de vila, desenvolvido em **Python** utilizando **Py5Script** (p5.js). 
O cenário consiste em uma vila habitada por um coelhinho, um riacho com peixes, vegetação interativa e um ciclo dinâmico de iluminação.
Link para a Vila 3D: https://esperanc.github.io/Py5Script/view.html?sketch=https://raw.githubusercontent.com/carolinafernandezds/Trabalho_Final_CG_Carolina_Falcao/refs/heads/main/vilinha_3d.py

## 🎮 Como Jogar / Interagir

A maquete é totalmente interativa! Para explorar a vila, siga os passos abaixo:

1. **Ative a tela:** Clique uma vez em qualquer lugar dentro da tela do jogo para que ela capture os comandos do seu teclado.
2. **Ande com o Coelhinho:** Use as **Setas do Teclado** (`↑`, `↓`, `←`, `→`) para caminhar pela vila. 
3. **Controle o Tempo (Dia/Noite):** Na **lateral direita**, clique e arraste o pino deslizante (scroll) para cima ou para baixo. Isso altera o clima, fazendo o Sol se pôr e as luzes quentes da vila se acenderem.
4. **Zoom da Maquete:** Na **lateral esquerda**, clique nos botões **`+`** e **`-`** para aproximar ou afastar a câmera do diorama.
5. **Gire o Mundo:** Abaixo do Zoom (canto inferior esquerdo), clique nas **setinhas do joystick virtual** para rotacionar a maquete 360 graus e explorar todos os ângulos das casas e do rio.
